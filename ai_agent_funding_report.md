# AI Agent 赛道近一年融资情况调研报告

> 数据采集时间：2024–2026 | 样本覆盖：6 家公司、9 条融资记录 | 报告日期：2025年6月

---

## 摘要

过去12个月内，AI Agent 赛道呈现出**“底座层极度吸金、应用层快速爆发”**的双轮驱动格局。全球头部大模型公司（OpenAI、Anthropic）单笔融资已突破千亿美元量级，而垂直应用侧的 Agent 企业（如 Cognition AI、CrewAI）也在短期内完成多轮跃升，估值增速远超传统 SaaS。

中国市场中，月之暗面（Moonshot AI）和智谱AI（Zhipu AI）分别完成累计超 12 亿美元和 20 亿人民币的融资，标志着国产 AI Agent 赛道进入**规模化资本竞逐阶段**。本报告基于 9 条核心融资记录，从融资规模、估值变化、投资方结构及趋势洞察四个维度进行深度拆解。

---

## 一、赛道概览：整体融资趋势与规模

### 1.1 全球融资总量与节奏

从 2024 年底至 2026 年初，样本内 6 家公司在不到 18 个月内共完成 **9 轮融资**，合计金额约 **1,563 亿美元**（不含 CrewAI 的 1,800 万美元，未统计智谱AI 20 亿人民币约 2.8 亿美元）。

其中，OpenAI 的两笔战略融资（400 亿 + 1,100 亿美元）与 Anthropic 的 Series G（300 亿美元）三家合计即达到 **1,800 亿美元**，占据全球 AI 融资总盘子的绝对大头。这反映出 AI Agent 赛道的资本集中度极高——**头部三家拿走了 90% 以上的资金**。

### 1.2 融资轮次分布

| 轮次类型 | 笔数 | 代表公司 | 平均单笔金额 |
|---------|------|---------|------------|
| 战略融资 | 2 | OpenAI | 750 亿美元 |
| Series G/D | 2 | Anthropic、Cognition AI | 155 亿美元 |
| Series A–C | 3 | CrewAI、Cognition AI、月之暗面 | ~1.7 亿美元 |
| C/D 轮 | 2 | 月之暗面 | ~6 亿美元 |
| 上市前最后一轮 | 1 | 智谱AI | ~20 亿人民币 |

> **洞察**：AI Agent 赛道的融资周期被显著压缩。Cognition AI 在 8 个月内完成从 Series C（4 亿美元）到 Series D（10 亿美元+）的跃升，月之暗面两轮间隔仅 2 个月。这种“超加速”节奏表明，投资方对 Agent 赛道的时间窗口判断极为紧迫，**不愿意给竞争对手留出窗口期**。

## 二、重点公司融资详情

### 2.1 OpenAI：全球最大 AI Agent 融资机器

| 指标 | 数据 |
|------|------|
| 融资轮次 | 战略融资（2 笔） |
| 融资金额 | 400 亿美元（2025-03）+ 1,100 亿美元（2026-02） |
| 投后估值 | 3,000 亿 → 8,400 亿美元 |
| 主要投资方 | SoftBank（700 亿）、Amazon（500 亿）、Nvidia（300 亿） |
| 估值增幅 | **180%**（18 个月） |

OpenAI 的融资结构体现了**“产业资本主导”**的鲜明特征：Amazon、Nvidia、SoftBank 三家产业巨头合计出资 1,500 亿美元，几乎没有传统 VC 的身影。这种“战略投资”逻辑并非单纯财务回报驱动，而是**生态卡位**——Amazon 需要 Agent 能力强化 AWS，Nvidia 需要 Agent 拉动算力消耗，SoftBank 则需要下一个“阿里级”资产。

### 2.2 Anthropic：Claude Code 定义 AI 编程 Agent

| 指标 | 数据 |
|------|------|
| 融资轮次 | Series G |
| 融资金额 | 300 亿美元 |
| 投后估值 | 3,800 亿美元 |
| 投资方 | GIC、Coatue（联合领投）；D.E. Shaw、Dragoneer、Founders Fund、ICONIQ、MGX（联合领投）；Accel、BlackRock、Sequoia、Temasek 等 |
| 核心产品 | Claude 系列大模型、Claude Code（AI 编程 Agent） |

Anthropic 的投资方名单几乎涵盖了**全球顶级主权基金、对冲基金、VC 和 PE 的全明星阵容**。值得注意的是，GIC（新加坡主权基金）和 Temasek 的参与，表明 AI Agent 已被主权资本视为**国家战略级资产**。Claude Code 作为“领先的 AI 编程 Agent”，其商业化路径（企业级代码助手）可能是本轮估值的支撑逻辑。

### 2.3 Cognition AI：Devin 的极速估值跃升

| 指标 | Series C（2025-09） | Series D（2026-05） |
|------|-------------------|-------------------|
| 融资金额 | 4 亿美元 | 10 亿美元+ |
| 投后估值 | 102 亿美元 | 260 亿美元 |
| 领投方 | Founders Fund | Lux Capital + General Catalyst + 8VC（联合领投） |
| 估值增幅 | — | **155%**（8 个月） |

Cognition AI 是全球**首个自主 AI 编程 Agent Devin** 的开发商，其估值增速（8 个月 155%）在整个赛道中极为罕见。关键数据支撑：

- **ARR 达 4.92 亿美元**——说明 Devin 已实现显著商业化
- **89% 代码由 AI 自行编写**——不仅是工具，而是替代人类工程师的“生产力重构”

从投资方变化来看，Series C 由 Founders Fund 领投（典型的早期激进 VC），到 Series D 引入 General Catalyst（更偏成长期），表明 Cognition AI 正在**从“技术验证”阶段快速过渡到“规模化扩张”阶段**。

### 2.4 CrewAI：多 Agent 协作框架的“小步快跑”

| 指标 | 数据 |

CrewAI 是样本中**唯一一个纯 Agent 框架层公司**，其 1,800 万美元的融资额远低于大模型底座公司，但运营数据亮眼：月执行超千万任务。Insight Partners 的进入表明，**Agent 编排层（Orchestration Layer）正在获得机构资本认可**，其商业模式更接近“下一代企业服务中间件”。

### 2.5 月之暗面（Moonshot AI）：国产 AI Agent 的资本标杆

| 指标 | C 轮（2025-12） | D 轮（2026-02） |
|------|----------------|----------------|
| 融资金额 | 5 亿美元 | 7 亿美元+ |
| 投后估值 | 43 亿美元 | 100 亿美元+ |
| 投资方 | IDG（领投）；阿里、腾讯等老股东超额认购 | 阿里巴巴、腾讯、五源资本、九安医疗等原有股东联合领投 |
| 估值增幅 | — | **133%**（2 个月） |

月之暗面是中国 AI Agent 赛道**融资节奏最快、估值跃升最陡峭**的公司。两轮累计融资超 **12 亿美元**，现金储备超 **100 亿人民币**。投资方几乎全为**中国互联网巨头和本土顶级 VC**（阿里、腾讯、IDG、五源资本），没有海外主权基金或美元 LP 参与，说明其融资路径具有鲜明的**“中国本土闭环”**特征。

2 个月内估值翻倍，反映了国内资本对 AI Agent 窗口期的焦虑——**宁可在高估值抢筹，也不愿错过下一个超级平台**。

## 三、关键发现与趋势分析

### 3.1 大模型底座 vs Agent 应用：资本的天平在哪？

| 维度 | 大模型底座（OpenAI/Anthropic/智谱） | Agent 应用（Cognition/CrewAI） |
|------|--------------------------------------|------------------------------|
| 单笔融资规模 | 百亿–千亿美元 | 千万–十亿美元 |
| 投资方类型 | 产业巨头、主权基金、国资 | 成长期 VC、产业天使 |
| 估值逻辑 | 技术壁垒 + 生态控制 | 商业化速度 + 替代效率 |
| 估值增速 | 稳定高基数增长 | 爆发式倍增 |

**结论**：资本并未在“底座”和“应用”之间二选一，而是**分层配置**。底座层吃掉了绝大部分资金（总量 90%+），但 Agent 应用层的估值增速（Cognition 155%、月之暗面 133%）远超底座层。这意味着，**Agent 应用可能是当前风险收益比最高的切入点**。

### 3.2 中美对比：两条路径，两个生态

| 维度 | 美国 | 中国 |
|------|------|------|
| 头部公司 | OpenAI、Anthropic、Cognition AI | 月之暗面、智谱AI |
| 单笔最大融资 | 1,100 亿美元（OpenAI） | 7 亿美元+（月之暗面） |
| 投资方结构 | 产业巨头 + 全球主权基金 + 顶级 VC | 互联网巨头 + 本土 VC + 政府引导基金 |
| 融资节奏 | 相对有序，战略投资为主 | 极度紧凑，抢窗口特征明显 |
| 政策因素 | 市场化主导 | 国资深度参与，上市路径明确 |

**核心差异**：美国市场的 AI Agent 融资是**全球化资本配置**（SoftBank、GIC、BlackRock 全球布局），而中国更多是**本土巨头和国资的“合纵连横”**。智谱AI 的港股上市和国资领投，标志着中国 AI Agent 赛道已经进入**“国家战略基础设施”**叙事。

### 3.3 估值增速：Agent 赛道的“非线性”特征

在样本中，有 3 家公司在短周期内实现了估值翻倍：

| 公司 | 时间周期 | 估值增幅 | 驱动因素 |
|------|---------|---------|---------|
| OpenAI | 18 个月 | +180% | 产品矩阵扩展 + 产业巨头加注 |
| Cognition AI | 8 个月 | +155% | ARR 爆发 + AI 自主编程能力验证 |
| 月之暗面 | 2 个月 | +133% | 国内巨头抢筹 + Agent 产品迭代 |

> **关键洞察**：AI Agent 赛道的估值增长并非线性，而是**事件驱动型跃迁**——一次产品突破、一轮巨头加注、一个商业化里程碑，都可能触发估值的阶跃式提升。这与传统 SaaS 的“ARR 线性估值模型”有本质区别。

### 3.4 投资方格局：从财务投资到生态卡位

本轮 AI Agent 融资的最大变化是：**传统财务 VC 的份额在萎缩，产业资本和战略投资者占据主导**。

- **OpenAI**：Amazon + Nvidia + SoftBank（产业巨头）
- **Anthropic**：GIC + BlackRock + 主权基金（国家战略资本）
- **月之暗面**：阿里 + 腾讯（互联网生态巨头）
- **智谱AI**：中关村科学城等（政府引导基金）

这意味着，AI Agent 赛道的投资逻辑已经从**“看财务回报”**转向**“看生态控制力”**。对于没有产业背景的纯财务投资者，进入门槛正在快速升高。

## 五、结论与展望

### 5.1 核心结论

1. **AI Agent 赛道已进入“资本军备竞赛”阶段**：全球头部公司在 18 个月内吸金超 1,500 亿美元，资本集中度创历史新高。

2. **产业资本主导，财务 VC 边缘化**：Amazon、Nvidia、阿里、腾讯等生态巨头以战略投资形式深度绑定 Agent 公司，纯财务投资者的入场门槛急剧升高。

3. **中美路径分化明显**：美国以全球化产业资本驱动，中国以本土互联网巨头 + 国资双轮驱动，且政策导向特征更强。

4. **Agent 应用层估值增速超底座层**：Cognition AI（155%）、月之暗面（133%）的估值跃升速度表明，**垂直 Agent 应用可能是当前风险收益比最高的投资切入点**。

### 5.2 未来展望

- **2026 下半年**：预计将出现更多“Agent + 行业”的垂直融资案例（法律、医疗、金融），底座层的千亿美元级融资可能放缓，但 Agent 应用层的并购整合将加速。
- **中国市场的关键变量**：智谱AI 的港股上市路径已为其他国产大模型公司提供了模板，预计 2026 年将有更多中国 AI Agent 公司启动 IPO 进程。
- **风险提示**：当前估值水平已高度透支未来 3–5 年的商业化预期，若 Agent 产品的实际采用率或变现效率不及预期，赛道可能面临阶段性估值回调。
